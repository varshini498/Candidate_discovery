# Intelligent Candidate Discovery

**Intelligent Candidate Discovery** is an AI-powered recruiter copilot that intelligently ranks candidates for a given job description using semantic understanding, skill matching, experience analysis, education, certifications, activity signals, and explainable AI. Instead of relying solely on keyword matching, the system understands the context of job requirements and compares them with candidate profiles to identify the best-fit talent.

Built as a modern **Streamlit** application, it automatically analyzes job descriptions, evaluates candidates using a multi-signal scoring engine, generates hiring recommendations with confidence scores, identifies skill gaps, creates AI-powered resume summaries and interview questions, and provides an interactive recruiter dashboard with candidate comparison, search, chat assistance, and PDF/Excel report generation.
